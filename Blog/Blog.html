React

Que es?
React es una biblioteca de JavaScript de código abierto diseñada para crear interfaces de usuario interactivas y eficientes. Gracias a su arquitectura basada en componentes, permite reutilizar código, optimizando el desarrollo y la escalabilidad de aplicaciones web.
Por que es popular?
Esto es debido a la amplia adopción en la industria y su comunidad activa que brindan acceso a recursos, soporte y numerosas oportunidades laborales. Además, los desarrolladores con experiencia en esta biblioteca (framework) son altamente demandados por empresas tecnológicas, ya que es clave en la creación de aplicaciones dinámicas y escalables.
Que son sus componentes?
Para crear una aplicacion en react se debe construir con componentes reutilizables que pueden ser combinados para formar interfaces complejas. Cada componente representa una parte de la interfaz de usuario y puede contener su propio estado y lógica. Estados de sus componentes
En React se gestiona el estado de los componentes a través de hooks, los cuales permiten actualizar y responder a cambios en la UI de manera eficiente. Entre los más utilizados están useState y useEffect
React con TypeScript y su importancia para la seguridad
TypeScript es una excelente herramienta para mejorar la seguridad del código en proyectos React, ya que agrega tipado estático y ayuda a evitar errores antes de ejecutar la aplicación.
Tipado Estricto
Activar el modo estricto en tsconfig.json mejora la seguridad al exigir un tipado más preciso
ejemplo:
{
"compilerOptions": {
"strict": true,
"noImplicitAny": true,
"strictNullChecks": true
}
}

¿Por qué es útil?


"noImplicitAny" evita variables sin tipo definido.
"strictNullChecks" previene errores de null y undefined.
